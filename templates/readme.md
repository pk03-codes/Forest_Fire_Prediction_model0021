# 🔥 Forest Fire Prediction Web App

## 📌 Overview
This project predicts Fire Weather Index (FWI) using machine learning and deploys it using Flask.

## 🚀 Features
- Data Cleaning & EDA
- Multiple ML Models (Linear, Ridge, Lasso)
- Model Evaluation
- Flask Web App
- REST API (/predict_api)
- Animated UI

## 🧠 Tech Stack
- Python
- Scikit-learn
- Flask
- HTML, CSS





## 📊 Model Used
Ridge Regression
